# 🚗 汽车报价单工具

一款专为汽车销售顾问设计的在线报价工具，支持新能源车和汽油车补贴计算，数据自动同步至飞书多维表格。

## 功能特点

### 📊 智能报价计算
- 支持**新能源车**和**汽油车**两种车型
- 自动计算各类补贴：
  - 国补（报废更新）：最高 12%/2万元
  - 省补（置换更新）：最高 8%/1.5万元
  - 地补：2k-4k（可按需调整）
- 实时计算购置税（新能源半税）
- 自动汇总开票价、政策补贴、最终落地价

### 💾 数据保存与同步
- 本地历史记录保存（LocalStorage）
- 一键回填历史报价
- 飞书机器人实时推送通知
- 飞书多维表格数据同步（需配置 Worker）

### 📋 其他功能
- 一键复制完整报价单
- 响应式设计，支持手机/平板/电脑
- 客户信息加密保存

## 补贴政策说明（2026年参考）

| 补贴类型 | 新能源车 | 汽油车 |
|---------|---------|--------|
| 国补（报废） | ≤12%（最高2万） | ≤10%（最高1.5万） |
| 省补（置换） | ≤8%（最高1.5万） | ≤6%（最高1.3万） |
| 地补 | 10万以下2000 / 10万以上4000 | - |

> 注意：购置税计算方式：开票价 ÷ 11.3 ÷ 2（新能源减半）

## 使用指南

### 1. 基础报价
1. 选择车辆类型（新能源/汽油车）
2. 选择补贴类型
3. 填写各项费用
4. 系统自动计算最终落地价

### 2. 保存报价
1. 点击「保存报价」按钮
2. 填写客户姓名、电话、车型
3. 点击「确认保存」
4. 数据自动保存至本地并同步飞书

### 3. 回填历史
1. 在「历史报价」区域点击任意记录
2. 系统自动回填所有数据
3. 修改后重新保存即可

## 自定义配置

### 修改补贴规则
编辑 `index.html` 中的 `calcSubsidy()` 和 `calcTax()` 函数：

```javascript
// 补贴比例修改示例
case 'scrap': 
  sub = isEv ? Math.min(invoice * 0.12, 20000) : Math.min(invoice * 0.10, 15000); 
  break;

// 购置税修改示例
return isEv ? Math.round(invoice / 11.3 / 2) : Math.round(invoice / 11.3);
```

### 修改飞书配置
1. **机器人 Webhook**：修改 `WEBHOOK_URL` 变量
2. **多维表格**：部署 Cloudflare Worker 并修改 `workerUrl` 变量

### 部署到 GitHub Pages
1. Fork 本仓库
2. 在 Settings → Pages 中启用
3. 等待自动部署完成
